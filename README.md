# leafletjs-dengan-codeigniter
penggunaan mapping geografis dengan framework leafletjs dan framework backend codeigniter

install codeigniter terlebih dahulu dan atur setting configurasinya seperti controller Home.php dam Model M_tps, 
dan jangan lupa download framework leafletjs seperti cluster, search, geojson,heatmap, Routing, semua itu bisa di dapatkan di websitenya Leafletjs. dengan melihat example dan beberapa downloadnya
